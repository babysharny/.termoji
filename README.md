## Install

### Download termoji to your home directory
```
cd ~
git clone https://github.com/babysharny/.termoji.git
```

### Add this code to your `.bashrc` or `.bashconfig` file
```
if [ -f ~/.termoji/main.sh ]; then
    source ~/.termoji/main.sh
fi
```

### Reboot your bash
```
source .bashrc
```
